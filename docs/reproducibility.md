# Reproducibility Guide

## Dataset: Derived demographic indicators for Latin America (2000-2023). CC0 1.0

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/indicadores-demograficos-latam-2000-2023.git
cd indicadores-demograficos-latam-2000-2023
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
