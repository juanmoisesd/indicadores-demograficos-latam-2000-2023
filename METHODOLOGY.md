## Abstract

Abstract (English)
&nbsp;
This derived dataset extends the original population dataset by age groups in Latin America (de la Serna, 2026; DOI: 10.5281/zenodo.18891177) by adding five calculated demographic indicators for 11 Latin American countries across four reference years (2000, 2010, 2020, and 2023). No raw data were modified; all variables were computed from the absolute population estimates already contained in the source dataset.
&nbsp;
The five indicators provided are: (1) Aging Index (AI): ratio of population aged 65+ to population aged 0-14, multiplied by 100; (2) Total Dependency R

## Data Collection Protocol

### Phase 1 — Search
Systematic search conducted in international databases including WHO, PAHO, World Bank,
ECLAC and national institutional repositories. AI-assisted search (Perplexity AI) with
human review and validation of all values.

### Phase 2 — Extraction
Data extracted and harmonized using Python (pandas). Country codes standardized to
ISO 3166-1 alpha-2. Temporal coverage aligned to annual frequency where available.

### Phase 3 — Validation
All values verified against primary sources. Inconsistencies flagged and corrected.
Data quality scores assigned (high/medium/low) based on source reliability and completeness.

### Phase 4 — Publication
Dataset deposited in Zenodo with DOI under CC0 1.0 license.
Metadata registered with DataCite and indexed in OpenAlex.
